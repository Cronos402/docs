## About Cronos402

Cronos402 is the first MCP payment gateway built exclusively for the Cronos blockchain, enabling AI agents and applications to make real-time payments using USDC.e (gasless) or native CRO.

## Built For

- **Cronos Labs Hackathon 2026** - Building the future of AI payments on Cronos
- **Ecosystem Integration Track** - Deep integration with Cronos blockchain
- **x402 Protocol** - Implementing the Payment Required standard on Cronos

## Key Features

- **Gasless USDC.e Payments** - Via official Cronos facilitator (EIP-3009)
- **Native CRO Support** - Direct wallet transactions
- **Cronos-Only Focus** - Purpose-built for the Cronos ecosystem
- **MCP Compatible** - Works with Claude Desktop, Cursor, and other MCP clients

## Technology Stack

- **SDK**: TypeScript SDK for building paid MCP servers
- **CLI**: Command-line tool for AI agents
- **Facilitator**: Leverages Cronos Labs x402 facilitator
- **Protocol**: x402 "Payment Required" standard
