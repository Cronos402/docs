The Cronos402 SDK provides everything you need to build and consume paid MCP servers on Cronos.

**Package**: [npmjs.com/package/cronos402](https://www.npmjs.com/package/cronos402) | **GitHub**: [Cronos402/sdk](https://github.com/Cronos402/sdk)

## Installation

```bash
npm install cronos402
```

## Main Exports

```typescript
import {
  // Server creation
  createMcpHandler,
  createBaseMcpHandler,
  startStdioServer,

  // Client
  withX402Client,

  // Network constants
  CRONOS_NETWORK,
  CRONOS_CHAIN_ID,
  CRONOS_RPC_URL,
  USDC_ADDRESS,
  CRO_ADDRESS,
  CRONOS_FACILITATOR,

  // Facilitator client
  CronosFacilitatorClient,
  createCronosFacilitator,

  // Payment utilities
  decodeCronosPayment,
  encodeCronosPayment,
  createCronosPaymentHeader,

  // Signers
  createSigner,
  createSignerFromViemAccount,
  cronosTestnet,
  cronos,

  // Proxy & hooks
  withProxy,
  X402MonetizationHook,
  LoggingHook,
  AnalyticsHook
} from "cronos402";
```
